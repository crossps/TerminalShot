# Changelog

## 1.0.0 — 2026-07-18

- Public release under the TerminalShot name.
- Full-page, area, window, full-screen, and manual scrolling capture.
- Adaptive scrolling and static-sidebar-aware stitching.
- Floating drag-out capture card.
- Vector annotation editor with crop, pixelation, step markers, and undo/redo.
- Per-user JSON settings and local-only capture storage.
